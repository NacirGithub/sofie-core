Additional Software & Hardware

Prerequisites#

  • Installed and running Sofie Core
  • Installed Playout Gateway
  • Installed and configured Studio Blueprints

The following pages are broken up by equipment type that is supported by Sofie's Gateways. A list of all the supported hardware and software is in the table below.

{% tabs %} {% tab title="Playout & Recording " %}

  • CasparCG Graphics and Video Server - Graphics / Playout / Recording
  • Blackmagic's Hyperdeck - Recording
  • Quantel Solutions - Playout
  • Vizrt Graphics Solutions - Graphics / Playout {% endtab %}

{% tab title="Vision Mixers" %}

  • Blackmagic's ATEM hardware vision mixers
  • vMix software vision mixer (coming soon) {% endtab %}

{% tab title="Audio Mixers" %}

  • Sisyfos audio controller
  • Lawo sound mixers, using emberplus protocol
  • Generic OSC (open sound control) {% endtab %}

{% tab title="PTZ Cameras" %}

  • Panasonic PTZ cameras {% endtab %}

{% tab title="Lights" %}

  • Pharos light control {% endtab %}

{% tab title="Other" %}

  • Generic OSC (open sound control)
  • Generic HTTP requests (to control http-REST interfaces)
  • Generic TCP-socket {% endtab %} {% endtabs %}
